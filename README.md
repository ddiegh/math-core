## Introducción
 **Math_core** es una biblioteca de analisis numerico creada solo con python (sin bibliotecas externas).

---
## Características

- Base: raices, linespace, arange, evualar polinomios

- Álgebra lineal: matrices, vectores, descomposición QR, triangularización gaussiana, calculo de eigenvalores

- Ecuaciones diferenciales ordinarias: metodo de euler

- Regresión lineal y polinomial por mínimos cuadrados

- Interpolación polinomial mediante matrices de Vandermonde
